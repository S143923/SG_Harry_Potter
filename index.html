Network Dimensions Over Time

Undirected networks have been constructed using the movie manuscripts, with character information collected from the Harry Potter Wiki, like the characters house, blood status and loyalties. Each movie script was divided into individual scenes and an edge was added between each character that occurred in the same scene. The network is based on all mentioned characters as the scripts werenâ€™t structured uniformly which made it too impractical to differentiate between Characters that talk in a scene, characters that only are mentioned in speech, or characters that are mentioned in stage directions. The size of the nodes is scaled by the degree of each node weighted with the number of scenes the character appears in. The number of nodes and edges for each of the movies are illustrated in the graph below

The high number of nodes in Deathly Hallows Part 1, is likely due to the requirement to connect all the loose ends in the story. The book with the minimum number of nodes and edges is Goblet of Fire, and is likely due to the very centered storyline around the Triwizard Tournament. This tournament keeps the story focused on the school and the people present, and this changes as the later book introduce outside groups such as the Order of the Phoenix.

Degrees Over Time

In the graphs below, the characters are evaluated in respect to the number of edges connected to each character, and as the networks are undirected it is not concerned with the nature of the edges. They are sorted according the total number of nodes across all movies, and the graphs illustrate the size of the size of the node in each movie. The main three characters Harry Potter, Ron Weasley and Hermione Granger are the 3 largest nodes in the respective order. The graphs not only indicate the level of involvement each character has in each book, but also (for the smaller characters) which movies they appear in.
